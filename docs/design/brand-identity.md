## 브랜드 핵심 이미지

### 공통 인상

이 기업의 브랜드 정체성은 다음과 같은 핵심 이미지로 구성됩니다:

- **실제 매장 / 실제 물량 / 실제 유통**
- **소규모 감성 X → 도매·공급·물류 기반**
- **꾸밈보다 정돈된 실용성**
- **"싸다"보다는 "많고 신선하고 바로 가져온다"**

### 색·공간 특징

- **메인 컬러**: 그린 계열 (신선, 농산물, 식자재)
- **조명**: 밝고 차가운 화이트 (청결·신선 강조)
- **상품 노출 방식**: 진열 미학보다 물량감 + 정렬

### 브랜드 포지션

```
감성 커머스 UI ❌
실물 기반 B2C·B2B 중간 포지션 UI ⭕
```

## 디자인 방향

### 한 줄 요약

> **"현장감을 살린 신뢰 중심 유통 플랫폼 UI"**

### 핵심 원칙

이 회사의 디자인은 **"예쁘게 보이는 쇼핑몰"**이 아니라  
**"실제 식자재 유통 현장을 웹으로 옮긴 플랫폼"**처럼 만들어야 합니다.

## 컬러 시스템

### 🎨 컬러 팔레트

| 역할 | 컬러 | HEX | 용도 |
|------|------|-----|------|
| **Primary** | Fresh Green | `#1E7F4F` ~ `#2E9F6B` | 메인 브랜드 컬러, 신선함 강조 |
| **Secondary** | Dark Green | `#0F3D2E` | 강조, 깊이감 |
| **Accent** | Fresh Yellow | `#F2C94C` | 공동구매/혜택 강조, CTA |
| **Background** | Off White | `#F7F8F7` | 배경, 여백 |
| **Text** | Dark Gray | `#1F2933` | 본문 텍스트 |

### 컬러 선택 이유

- **그린**: 신선 / 농산물 / 식자재의 상징
- **노랑**: 가격·혜택·공동구매 CTA 강조
- **오프화이트**: 청결하고 깔끔한 느낌
- **진한 그레이**: 가독성과 신뢰감

### 사용 가이드

```css
/* Tailwind CSS Custom Colors */
--color-primary: #1E7F4F;
--color-primary-light: #2E9F6B;
--color-secondary: #0F3D2E;
--color-accent: #F2C94C;
--color-background: #F7F8F7;
--color-text: #1F2933;
```

## 타이포그래피

### 🔤 폰트 시스템

| 요소 | 폰트 | 굵기 | 크기 | 용도 |
|------|------|------|------|------|
| **H1** | Pretendard / SUIT / Noto Sans KR | Bold | 2xl~3xl | 페이지 제목 |
| **H2** | Pretendard / SUIT / Noto Sans KR | SemiBold | xl~2xl | 섹션 제목 |
| **가격·수량** | Pretendard / SUIT / Noto Sans KR | SemiBold | lg~xl | 가격, 수량 정보 |
| **본문** | Pretendard / SUIT / Noto Sans KR | Regular | base | 설명 텍스트 |
| **CTA** | Pretendard / SUIT / Noto Sans KR | SemiBold | base~lg | 버튼 텍스트 |

### 타이포그래피 원칙

- ✅ **가독성 = 신뢰**: 명확한 굵기 분리
- ✅ **장식 폰트 사용 금지**: 실용성 우선
- ✅ **일관된 폰트 패밀리**: 브랜드 통일성

### 사용 예시

```tsx
// 제목
<h1 className="font-bold text-2xl text-gray-900">상품 목록</h1>

// 가격
<p className="font-semibold text-lg text-gray-900">12,900원</p>

// 본문
<p className="font-normal text-base text-gray-600">신선한 식자재를 공동구매로 더 합리적으로</p>
```

## 레이아웃 구조

### 🧱 메인 페이지 구조

```
┌─────────────────────────────────────┐
│  [ Hero Section ]                   │
│  - 실제 매장 사진 (풀 width)        │
│  - 문구: "신선한 식자재,            │
│    공동구매로 더 합리적으로"         │
│  - CTA: 공동구매 보기 / 오늘의 특가 │
├─────────────────────────────────────┤
│  [ 핵심 서비스 요약 ]               │
│  - 공동구매                          │
│  - 대량 유통                         │
│  - 당일/근거리 공급                  │
├─────────────────────────────────────┤
│  [ 공동구매 상품 리스트 ]            │
│  - 카드형이지만 밀도 높게            │
│  - 수량/잔여량/마감 시간 강조       │
├─────────────────────────────────────┤
│  [ 신뢰 요소 ]                       │
│  - 실제 매장 사진                    │
│  - 유통 과정 요약                    │
│  - 취급 품목군                       │
├─────────────────────────────────────┤
│  [ Footer ]                         │
│  - 매장 정보                         │
│  - 유통 파트너                       │
└─────────────────────────────────────┘
```

### 레이아웃 원칙

1. **Hero Section**: 실제 매장 사진으로 현장감 강조
2. **서비스 요약**: 핵심 가치 명확히 전달
3. **상품 리스트**: 밀도 높은 카드형 레이아웃
4. **신뢰 요소**: 실제 물량, 유통 과정 시각화
5. **Footer**: 실무 정보 중심

## 상품 카드 디자인

### ✅ 추천 카드 구조

```
┌──────────────────┐
│  [이미지 영역]   │ ← 이미지 없어도 공간 유지
│                  │
├──────────────────┤
│ 상품명            │
│ 가격  12,900원    │
│ ███████░░ 75%     │ ← 진행률 (공동구매 핵심)
│ 잔여 25 / 100     │
└──────────────────┘
```

### 카드 디자인 원칙

#### ❌ 하면 안 되는 것

- 너무 둥근 카드 (감성 커머스 느낌)
- 그림자 과다 (불필요한 장식)
- 이미지 중심 감성 카드 (실용성 저하)

#### ✅ 추천 사항

- **밀도 높은 레이아웃**: 정보를 효율적으로 배치
- **진행률 강조**: 공동구매의 핵심 요소
- **수량 정보 명확**: 잔여량/목표량 표시
- **이미지 없어도 공간 유지**: 레이아웃 안정성

### 품절 상품 처리

- 카드 전체 채도 ↓ (opacity 감소)
- 이미지 위 SOLD OUT 오버레이
- 하단 고정 배치 (UX 요구사항 반영)

### 카드 스타일 예시

```tsx
// 일반 카드
<div className="border border-gray-200 rounded-lg overflow-hidden">
  <div className="aspect-square bg-gray-100">
    {/* 이미지 또는 placeholder */}
  </div>
  <div className="p-4">
    <h3 className="font-semibold text-sm">상품명</h3>
    <p className="font-semibold text-lg text-gray-900">12,900원</p>
    <div className="mt-2">
      <div className="w-full bg-gray-200 rounded-full h-2">
        <div className="bg-green-600 h-2 rounded-full" style={{ width: '75%' }} />
      </div>
      <p className="text-xs text-gray-600 mt-1">잔여 25 / 100</p>
    </div>
  </div>
</div>

// 품절 카드
<div className="border border-gray-200 rounded-lg overflow-hidden opacity-50 grayscale">
  {/* 동일 구조 + SOLD OUT 오버레이 */}
</div>
```

## 공동구매 UX 포인트

### 🔥 반드시 강조해야 할 요소

1. **현재 수량 / 목표 수량**
   - 진행률 바 시각화
   - 숫자로 명확히 표시

2. **마감 임박 여부**
   - 시간 카운트다운
   - 시각적 경고 (노란색/빨간색)

3. **성공 시 가격 혜택**
   - 원가 vs 공동구매 가격 비교
   - 할인율 강조

4. **실제 물량 기반이라는 신뢰**
   - 실제 매장 사진
   - 물량 정보 표시

### 시각적 우선순위

| 순위 | 요소 | 이유 |
|------|------|------|
| **1순위** | 가격 | 구매 결정의 핵심 |
| **2순위** | 진행률 | 공동구매 성공 여부 |
| **3순위** | 남은 수량 / 마감 시간 | 긴급성 강조 |
| **4순위** | 상품 이미지 | 보조 정보 |

### 일반 쇼핑몰과의 차이점

| 일반 쇼핑몰 | 공동구매 플랫폼 |
|------------|----------------|
| 상품 이미지 중심 | 가격·진행률 중심 |
| 개별 구매 | 집단 구매 |
| 즉시 구매 가능 | 목표 달성 시 구매 |
| 감성적 디자인 | 실용적 디자인 |

## 반응형 전략

### 📱 모바일

- **레이아웃**: 리스트형 (1열)
- **정보 배치**: 가격·진행률이 한 화면에 들어와야 함
- **CTA**: 버튼 항상 하단 고정 고려
- **터치 영역**: 최소 44px × 44px

### 💻 PC

- **레이아웃**: 3~4열 그리드
- **인터랙션**: 마우스 오버 시 상세 정보 노출
- **정보 밀도**: 모바일보다 더 많은 정보 표시 가능

### 반응형 브레이크포인트

```css
/* Tailwind CSS Breakpoints */
sm: 640px   /* 모바일 가로 */
md: 768px   /* 태블릿 */
lg: 1024px  /* PC */
xl: 1280px  /* 대형 PC */
```

### 그리드 시스템

```tsx
// 모바일: 1열
<div className="grid grid-cols-1 gap-4">

// 태블릿: 2열
<div className="md:grid-cols-2 gap-4">

// PC: 3~4열
<div className="lg:grid-cols-3 xl:grid-cols-4 gap-4">
```

- **Tailwind CSS + Design Token**

  - 그리드/반응형 최적화

## 디자인 키워드

이 기업에 어울리는 디자인 키워드 7가지:

1. **현장감** - 실제 매장, 실제 물량을 느낄 수 있는 디자인
2. **신뢰** - 투명한 정보, 명확한 가격, 실제 사진
3. **물량** - 대량 유통의 느낌, 정렬된 진열
4. **정렬** - 깔끔하고 정돈된 레이아웃
5. **신선** - 그린 컬러, 밝은 조명, 깨끗한 느낌
6. **가격 투명성** - 명확한 가격 표시, 할인율 강조
7. **실용성** - 장식보다 기능, 감성보다 정보

## 결론

### 디자인 철학

이 회사의 디자인은:

- ❌ "예쁘게 보이는 쇼핑몰"
- ✅ "실제 식자재 유통 현장을 웹으로 옮긴 플랫폼"

### 핵심 원칙

1. **현장감**: 실제 매장 사진, 실제 물량 정보
2. **신뢰**: 투명한 정보, 명확한 가격
3. **실용성**: 장식보다 기능, 감성보다 정보
4. **공동구매 특화**: 진행률, 수량, 마감 시간 강조

### 구현 우선순위

1. **1순위**: 상품 카드 디자인 (진행률, 수량 강조)
2. **2순위**: 컬러 시스템 적용 (그린, 노랑)
3. **3순위**: 타이포그래피 시스템 (가독성 중심)
4. **4순위**: 반응형 레이아웃 (모바일 1열, PC 3~4열)
