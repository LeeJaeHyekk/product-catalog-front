# 오류 문서화

이 디렉토리는 프로젝트에서 발생한 오류와 해결 방법을 문서화합니다.

## 오류 목록

### 1. Hydration Error
- **파일**: `hydration-error.md`
- **발생 시점**: 서버 렌더링과 클라이언트 하이드레이션 불일치
- **해결 상태**: ✅ 해결됨

## 오류 추가 방법

새로운 오류가 발생하면 다음 형식으로 문서를 작성하세요:

1. `docs/errors/[error-name].md` 파일 생성
2. 다음 섹션 포함:
   - 문제 발생 원인
   - 증상
   - 해결 방법
   - 예방 방법
   - 참고 자료

3. 이 README에 오류 목록 추가
