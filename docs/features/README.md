## 추가 기능 문서 작성 방법

새로운 기능이 추가되면 다음 형식으로 문서를 작성하세요:

1. `docs/features/[feature-name].md` 파일 생성
2. 다음 섹션 포함:
   - 구현 내용
   - 사용 방법
   - 장점
   - 예시 코드
3. 이 README에 기능 목록 추가
